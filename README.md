# Practice: Creating Arrays 2

## Description
This project focuses on advanced array manipulation in JavaScript. It explores initializing arrays with a specific size, using the `.fill()` method for bulk updates, and utilizing `for` loops to programmatically populate data.

## Challenges Completed

### Task 1: Initialize & Fill
* Created a new array with a fixed size of 7.
* Used `.fill("Hello")` to populate every slot.

### Task 2: Partial Fill
* Demonstrated the use of start and end parameters in `.fill()`.
* Updated a specific range of the array (e.g., indices 0-3) with a new string.

### Task 3: Loop Population
* Created an array of size 5.
* Implemented a `for` loop to assign numeric values based on the index formula: `index * 10`.
* Verified the output array: `[0, 10, 20, 30, 40]`.

## Technical Skills
* **JavaScript Constructor:** `new Array(size)`
* **Array Methods:** `.fill(value, start, end)`
* **Control Flow:** `for` loop iteration and arithmetic assignment.
